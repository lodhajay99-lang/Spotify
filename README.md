# Spotify Web Clone

A responsive, feature-rich web-based music player inspired by Spotify, built with HTML, CSS, and Vanilla JavaScript.

## 🎵 Features

- **Audio Playback:** Play, pause, skip forward, and skip backward.
- **Dynamic Search:** Search through the available song list with real-time filtering.
- **Playback Controls:** 
  - **Shuffle Mode:** Switch between Next, Repeat, and Random playback modes.
  - **Volume Control:** Adjust the volume smoothly with a slider, or mute/unmute the audio entirely.
  - **Seek Bar:** Jump to any specific part of a song with an interactive progress bar.
- **Visuals:** An active audio wave animation displayed while the music is playing, highlighting the current playing track.
- **Download Capability:** Download the currently playing song directly to your device.
- **Responsive Layout:** Well-structured UI elements including side navigation, play queues, and popular artists sections.

## 💻 Technologies Used

- **HTML5:** For the structural layout of the music player.
- **CSS3:** For styling, grid/flexbox layouts, animations, and custom scrollbars/sliders.
- **JavaScript (ES6):** For handling logic like DOM manipulation, audio playback APIs, event listeners, and dynamic UI updates.
- **Bootstrap Icons:** Used for intuitive scalable vector icons (play, pause, volume, search, etc.).

## 📂 Project Structure

- `index.html` - The main layout of the application.
- `Music.css` - Custom styling and animations.
- `app.js` - Contains all the application logic, event listeners, array of songs, and audio state management.
- `Img/` - Contains all the artist and song posters/images.
- `audio/` - Contains all the MP3 audio files.

## 🚀 Getting Started

To run the project locally on your machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/lodhajay99-lang/Spotify.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Spotify
   ```
3. Open `index.html` in your favorite web browser or use a tool like Live Server in VS Code to run it locally.
